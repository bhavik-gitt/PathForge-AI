## 📖 Overview

PathForge AI is an AI-powered career and learning mentor that generates personalized learning roadmaps based on a user's goals, current skills, available study time, and target timeline.

Instead of spending hours figuring out:

- What to learn
- In what order to learn
- Which projects to build
- How to prepare for interviews

PathForge AI creates a structured roadmap within seconds.

---

## 🎯 Problem Statement

Many students and aspiring developers struggle with:

- Information overload
- Unclear learning paths
- Lack of mentorship
- Skill-gap identification
- Project selection
- Interview preparation

PathForge AI solves these problems by acting as an intelligent career mentor.
